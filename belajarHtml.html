Pengenalan

HTML adalah bahasa markup standar untuk membuat halaman web. HTML adalah singkatan dari Hyper Text Markup Language.

Beberapa tag yang unik dan advance di HTML adalah: <canvas>, <video>, <audio>, <details>, <summary>, dan masih banyak lagi.
