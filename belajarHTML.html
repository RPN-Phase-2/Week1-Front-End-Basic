What is RPN?

RPN is free online code community based on discord, This community is build since July 2023. The main purpose of this community to make anyone can code regardless their background in IT or not. In this community is coming with 3 phase.

  • Phase 0: Learning about fundamental of Programming
  • Phase 1: Learning about Backend development
  • Phase 2: Learning about Frontend development
  • Phase 3: Learning about nowdays technology and your carrer path in future

TLDR:

RPN is online code community based on discord, mainly learning about website development
